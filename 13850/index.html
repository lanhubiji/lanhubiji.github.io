    收录于话题

    前一阵子一直很忙，没时间总结对微信小程序的理解，放假第一天，终于可把一些想法梳理出来。

    \

    一、长尾世界与微信小程序的完美契合

    \

    按照张小龙的说法：小程序是一种不需要下载安装即可使用的应用，它实现了应用“触手可及”的梦想，用户扫一扫或者搜一下即可打开应用。也体现了“用完即走”的理念，用户不关心是否安装太多应用的问题。应用将无处不在，随时随地可用，但又无需安装卸载。

    \

    基本上是站在用户的角度描述小程序的，这跟很多分析文章站在开发者和影响行业的角度不同，从用户的角度，小程序：

    1.无须安装即可使用

    2.扫一扫或搜一下就可打开应用

    3.用完即走

    4.不用担心安装太多应用

    5.无处不在，随时随地可用。

    \

    它满足用户的核心诉求：使用应用不再必须去应用商店下载、注册、登录，省去了一系列操作的麻烦。让使用应用变得简单方便。这个诉求的背后是什么？是需求的频次和刚需度。如果需求频次够高，需求度沟刚，对于用户来说，下载、安装、注册、登录都不是麻烦事。但如果是低频的需求，手机上有太多的应用，确实是鸡肋。用之不常，但删之可惜。比如对于一个不经常出差的人来说，手机安装一个携程APP订机票订酒店，安装一个航旅纵横选座或查看航班延误情况，确实有点多余。如用户只需用微信搜一搜或扫一扫等就可解决问题，为什么还要安装更多的APP呢？

    \

    小程序核心满足的是长尾需求。长尾虽然是小众，但抵不住人多，终究汇总成为一个巨大的需求。了解过长尾理论的人都知道，长尾是一个在信息时代才存在的现象。线下的书店由于货架是有限的，所以，永远都是在有限的货架上尽可能把可能畅销的书籍摆上，冷门的则无法上架。电影院排片也是一个道理，放映厅和时间都是有限资源，有些冷门影片永远无法公映。但在互联网上就不一样，由于可以搜一搜，可以在线分类，货架是无限的。很多冷门的书籍即使卖个十本八本，但抵不住购买的人多，无数的冷门书籍累计起来的销售量也是非常可观的。

    \

    而微信的小程序也是同样的道理。它就是要成为所有长尾应用的入口，这就是微信小程序未来威力最大的地方。

    \

    另外，从开发者的角度，也有更大的动力去做冷门或低频的应用。由于有聚合效应，降低了开发成本、获客成本，优质的小应用也有机会生存。

    \

    二、入口场景YY

    **
    **

    小程序入口在哪里？用户怎么实现触手可及，随时随地可用，用完即走的体验？目前透露的方式是通过搜一搜、扫一扫。可能的方式很多种：

    \

    方式一：在任何地方搜索时，不仅出现聊天记录列表、朋友圈、公众号、文章，还可出现应用。找到后点击即可使用。凡是使用过的应用都进入用户的微信列表，可能是单个形态出现，也可能是聚合在小程序集合中。

    \

    方式二：微信右上角“+”添加一个小程序入口，不过概率很小。

    \

    方式三：在通讯录下，增加一个“小程序”与“公众号”并列，作为曾经使用过的或者收藏的小程序入口。

    \

    方式四：“发现”TAB里面增加一个“小程序”入口，点击小程序会出现一个搜索框？还是其他？通过搜索推荐应用，点击应用即可使用，使用完之后进入微信列表及通讯录的“小程序”集合中？好处容易找到，不利的地方是路径略长。比如要订机票，点击发现TAB，然后点击“小程序”，再搜索“订机票”，然后再出来应用，再点击打开。如果“发现”TAB不仅有“搜索框”，还有应用的分类和汇总，这个貌似不符合微信的理念。按照微信的一贯的做法，它不会去做一个类似于app
    store一样的市场，不做中心化的聚合，而是让用户自发去传播。

    \

    方式五：入口无处不在，让开发者和忠实用户去推荐和传播？这符合微信的惯常做法。微信更可能通过聊天、朋友圈等去中心化等方式，让小程序可以在它需要出现的时候出现，通过用户自传播的方式去流通，当然也可通过搜一搜和扫一扫的方式主动获取。

    \

    那么，具体应用会出现在朋友圈信息流中？出现聊天界面？跟公众号文章一样，用户可以推荐，接受者看到后即可打开引用？比如你心血来潮，突然想找一位健身教练帮你健身塑型，刚好在朋友圈或者群里面看到朋友推荐的约教练应用，点击打开即可根据地理位置、特点、价格等自己需求去查找。甚至小程序本身就是一个超级公众号？用户关注之后，可以跟小程序进行聊天，用户输入文字或者语音，即可获得应用的推荐。比如当你用语音或者文字输入：本月订去伦敦的机票，怎么乘机最便宜？微信会智能推荐满足需求的最佳应用。即使在PC互联网时代，人们使用网站，很多时候也不是通过输入网址直达的，大多数人懒得去记或输入网址，尤其是非高频使用的网站，更多是愿意通过百度搜索网站名或者通过导航网站进入。按照人性的懒惰，对于在微信上获取应用也是同理，让用户直接搜索或者通过便利导航获取更方便。如何从概念转化为实际的产品形态，绝对是一件考验人的事情。期待微信团队有惊人亮眼的解决方案。

    \

    三、入口级新硬件来临之前，小程序进一步加强了垄断，形成一超多强局面

    \

    之前写过一篇关于《打败微信的将会是一款什么样的产品？》聊到打败微信的几个条件，如硬件入口的改变、代际更替与用户习惯变迁等。这些都是基于大环境级别的变化才能触发的。

    \

    微信通过高频和刚需的聊天、朋友圈、公众号、服务号以及微信支付等让它成为人们手机上超级应用，让绝大多数用户养成打开手机就下意识打开微信的习惯。或者反过来说，人们低头看手机是因为要打开微信。微信成为了整个中国移动互联网的最大入口。

    \

    而今天微信小程序的到来，不管是看好或者看坏，都不可否认这个重量级选手对整个移动互联网带来的影响。小程序毫无疑问将进一步加强流量垄断。跟之前公众号、服务号不同的是，这次的小程序，是应用，让用户体验更好的应用，是实现微信“连接一起”这个超级梦想的关键一步。

    \

     如果小程序的生态形成，人们手机里需要安装的应用将越来越少。当微信垄断了50%以上的移动互联网流量之后，一个超级垄断的信息形态就将诞生。这将是一个史无前例的流量集中，巨型的微信OS。

    \

    PC时代，搜索、即时通讯、内容、电商以及成千上万的中小网站共同掌握流量，虽然有几个超级大的流量节点，如搜索、电商和即时通讯，但还是一个相对分散的流量体系，无法达到如此高的垄断。

    \

    微信小程序生态能否成功的核心的关键是用户习惯。如果用户觉得超过10个APP是很麻烦的事情，在解决问题时能首先想到微信，而不是去搜索应用市场下载APP来解决，那么，这个梦想就很近了。

    \

    微信不再是聊天软件，微信已经是复合体。它不仅有聊天、群聊，更有朋友圈、公众号、服务号、微信支付、游戏、购物等。它在用户的心目中已经完成了心智认知的转变：微信已经是超级综合体。它变成了你想要干什么，它都可以满足你的一个认知。所以，通过微信上的应用来解决用户自身需求，并不存在认知上的障碍。

    \

    一系列的嵌入，一系列的成功，用户并没有因为微信增加了更多功能而逃离，反而是变得更加离不开。想想看，很多用户在线下支付时，首选的是微信支付，简单而方便。但如果你是卖沙发的，有一天你突然卖汽车零部件，用户会蒙圈。而在微信这个超级综合体，这个不是障碍。

    \

    说到底，微信的核心是沟通。这是人类的几大最核心需求之一。从工业时代开始，能够产生大企业的，要么是沟通，要么是交通出行，要么是能源，要么是信息…领域屈指可数。这些基本需求产生了石油大王、汽车大王、电信大王等。

    未来要颠覆这个超级巨无霸，只有一个稍纵即逝的机会，就是未来全新硬件平台的带来。这个平台必须是可用于沟通的，相当于历史上的电话、电视和广播、PC、智能手机级别的。如果这个硬件更替的时机不早点到来，那么，微信的超级流量垄断只会越来越强。其他人，甚至其他巨头都会绝望。

    \

    在新的入口级硬件出来之前，整个移动互联网的格局，更现实情况是出现一超多强的局面：一个超级生态帝国是微信，相当于鼎盛时期的罗马帝国。它除了垄断中国的即时沟通、朋友圈社交之外，还拥有支付、游戏、媒体平台、应用平台等。最终是连接一切适合它连接的东西。终极目标是整个世界的连接器。与此同时，在垂直领域有自己特色和护城河的APP也继续获得生存机会，毕竟高频和刚需产品已经形成了一定的用户习惯、有更好的用户体验，还有基础的数据积累，还是有护城河的。最后还会有一些小众APP，但流量越来越式微。移动互联网的流量格局，极有可能出现倒三角状态，顶部是微信，中间是一些有一定基础的APP，底部是越来越小众的APP。

    \
