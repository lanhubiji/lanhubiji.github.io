    功能介绍 通往区块链的新世界。

    **

    收录于话题

    （梵高，麦田里的收割者）

    ico持续火爆，浑水摸鱼者各种圈钱项目应接不暇，韭菜都不够割了。有的项目完全不靠谱，跟区块链半毛钱关系都没有，还拿来做ico，诸如此类的乱象很多。稍有常识的人都能分辨出来。

    \

    但，

    \

    可怕的是哪些跟区块链还有点关系，项目概念不错，团队背景也强大的项目，这些项目往往追随者众，大家挤破头去抢ico。

    \

    因为有的ico上线后能涨几倍、几十倍，所以即使大家都清楚现在泡沫很严重，还是会前仆后继进来。

    \

    从回报角度，这么也正常。

    \

    但其中ico的几个乱象值得警惕，容易把长期生态变成涸泽而渔，最终不利于区块链整体行业的发展。

    \

    乱象之一：有差价的pre-ico或预众筹

    \

    区块链的精神是分布式和去中心化，而pre-ico基本上把份额卖给了有关系的机构或者个人，相当于以较低的价格出售，这对于在公开市场参与ico的普通投资人来说是否公平？还没有上交易所就已经有大的套利空间了。

    \

    当然如果是公开市场ico的项目，有早鸟阶段，先到的有优惠，这个很正常，给更早决定的人以激励。但pre-ico的做法本质上是传统vc和投行的套路，跟区块链的精神是相违背的。

    \

    当然从收益角度，作为个人投资者有机会和资源参与pre-ico没有错，但本质上来说这不是区块链ico应该发展的方向，这也不利于行业的健康发展，会催生越来越多的圈钱操作和套利。

    \

    乱象之二：融资总额无上限和个人投资额不设限

    \

    像一些ico项目，如tezos等一些知名项目，不设上限，由于项目概念好，营销好，个人投资者疯狂涌入，造成项目还没有产品就估值几十亿甚至几百亿人民币，在ico领域诞生了不少没有产品的独角兽，这是传统创投行业经过多少年的血泪打拼才能拿到的估值啊。目前全世界的独角兽也不过几百个而已。现在的ico无上限项目秒变独角兽，这世道真是不一样了啊！

    \

    这跟传统的中心化公司一样的贪婪无度!而区块链项目的本质是要去公司化，变成分布式的社区化，且不说那个模式更好，这是不符合区块链精神和发展趋势的。

    \

    另外对个人投资者投资额没有限制、多多益善的项目也是不符合区块链去中心化和分布式的社区精神的。

    \

    那些富有的个人投资者通过购买大量的代币，后续可以操纵市场，这对于项目本身来说也没有好处。

    \

    区块链是去中心化和分布式本身社区化的网络。在众筹中对众筹额度不设上限，对个人投资者不设上限的都不是符合区块链精神的项目。不设投资额度上限让参与人数变少，不利于社区文化形成，越来越走向中心化。

    \

    现状是好些知名的项目都在搞预售，都在搞不设众筹上限，对个人投资者不做任何额度限制，这些都是区块链ico行业的乱象，是不利于行业发展的。

    \

    从目前回报看，那些没有节制滥发代币的项目回报率也不高。

    \

    对于真正想做事的区块链项目团队，建议尽量做符合区块链精神的事情：

    \

    1.少参与pre-ico，通过公开发行渠道给更多人以更合理的价格参与进来

    \

    2.对项目做合理估值，ico设置项目融资上限，给投资人合理的回报空间，利于后续项目获得更多的社区支持者，促进良性循环

    \

    3.对项目的每个投资人的投资额度做一定限制，让更多人参与进来，更多人获益，获得更多社区粉丝，真正推动项目和区块链的发展。

    \

    对于个人投资者，建议：

    \

    尽量不投不支持无融资上限的项目，这类项目上交易所后的回报率很难得到保障，很多历史项目已经证明，这里无须多说。

    \

    目前来看，一些融资有节制的项目，比如omisego等上交易所后的回报都是比较出色的。
