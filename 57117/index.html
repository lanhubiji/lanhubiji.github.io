    假如你是2009年开心网CEO，你会做得更好？

    收录于话题

     这两天开心网创始人程炳皓写了过去8年的总结和反思，在业界引来不大不小的讨论。

    \

    从他的反思里面能看到很多方面，有人说他就是一个土鳖工程师，有理想无格局，有做事韧劲无远大战略。有人说他实属不易，虽无法重现当日辉煌，也算是全身而退，可以为下一个篇章而努力。

    \

    不管你怎么评价，有个事实是毫无争议的：开心网曾经是中国互联网历史上红极一时的网站，它在中国互联网发展史上占有自己的一席之地。\

    \

    最终开心网没有持续它的辉煌，盛极而衰。它没能像facebook一样最终走向的更高的层面，没能成为互联网超级节点。\

    \

    先来一个假设，假如你是当年开心网CEO，当开心网爆发之后，你会怎么做？在夜深人静的时候，问问自己内心：你真的会比程炳皓做得更好吗？

    \

    你或许会发现如果你不是超级创造者或者超级好运气的创业者，你在这个位子上，做的决策不见得会比程炳皓更出色，更多的情况是会更差。

    \

    真正的超级创业者，是有真正的创业方法论，有深度思考的。比如美团的王兴，他对事情的理解深度在互联网大佬中也是罕见，很难看到其他大佬的公开发言有他的深度。如果你不是这一类的创业者或者运气超级好的创业者（决策碰巧蒙对的），其实不管是谁，站在程炳皓的位置上，都会面临艰难的决策。

    \

    从马后炮（也就是当时没有人能看得特别清楚，几乎所有人都是当局者迷）的分析看，有两个节点是关键，2009年最火的时候，没有提前为下一步做好布局或者延续的提升。2010年活跃下滑的时候，也没有找到真正的应对之策。与此同时，移动互联网的大潮来临，新的产品形态和需求的匹配方面也没有找到更多的突破，只是简单地按照原有的逻辑进行模仿，但没有真正的创造，比如模仿pinterest的品集，模仿path的美刻。

    \

    那么问题来了：

    • 回到2009年，假如你是开心网的CEO，这时候的开心网火得不要不要的，一时风光无两。那时还没有微博、微信，你觉得下一步应该怎么做？

    • 回到2010年，开心网活跃度大幅下滑，微博开始走红，微信初现端倪（2011年1月21日推出的），你觉得应该怎么做？如何应对？

    \

    如果不是马后炮的分析，你能有决断开发出适合移动互联网的通讯服务（类微信），你有决断在开心网流量上诞生出比微博更早的twitter类产品？

    \

    你会发现所有的这一切几乎不可能。最终还是回到一个原点：什么造就了你，什么就能让你坠落，或者什么能让你强者愈强。

    \

    如果没有极其激烈的外因或者本身具有极强领悟力去催化或蜕变，所有的成败都取决于一点：你的基因决定了你的认知模式和行为，结果（成功或失败）只是最终的表现而已。

    \

    下面来分析为什么。

    \

    你能在2009年就判断出熟人社交本身的问题吗？很难。

    \

    社交只有两种交互方式：一种是即时通讯，一种是延时交流（动态信息流）。即时通讯，比如QQ、微信、whatsapp、line等的通讯功能（点对点交流或群组交流），而动态信息流，比如QQ空间、微信的朋友圈等

    \

    从沟通本质上来讲QQ和微信满足的需求是一样的，但由于PC与移动终端的差异，它们的产品形态明显有不同。比如说QQ空间和微信朋友圈，QQ空间更重，是个人空间全方面展示，朋友圈更轻，以照片分享为主，更碎片化。微信的聊天更轻，组群更方便。但不管产品形态怎么变化，最后它满足的都是即时通讯和动态信息流的分享。

    \

    而开心网一开始满足的基于朋友（熟人）之间的社交游戏的需求，它本质上跟即时通讯和动态信息分享的出发点是不同的。（这一点从程炳皓8年的开心网反思也能看出来，开心网是因为什么成立，是因为什么火爆的）

    \

    换句话说，人们来到开心网核心目标不是要进行即时通讯或者动态信息分享。人们上ＱＱ，上微信，上what’sapp，上line，浏览QQ空间、发朋友圈、上facebook等，核心就是即时时通讯或朋友间信息流的分享。

    \

    这意味着开心网一旦把自己归入社交网络，对标facebook的时候，在认知上与用户期望产生了重大的错位，这个认知的错位，导致当它面对突然而来的幸福时，不知道该怎么驾驭。（注：这是马后炮，蓝狐在2009年2010年绝对不会有能力认识到这一点）

    \

    如果对标facebook，那么就会开发分享朋友圈信息的功能，但这并不是用户对开心网的认知和期待。

    \

    这样产生了矛盾，用户对开心网真正价值的认知和开心网对自己定位认知产生了差异，这个差异导致了接下来的决策的不同。导致最终的迷茫和错乱。

    \

    如果一开始就清晰地了解自己的基因，基于这一点去深挖掘，是不是能做到更好，也不得而知。但有一点，做社交网络，能成的概率很小，除非比别人提前布局，下最大的决断，倾全力去取做最好产品形态的移动通讯产品。但，这可能么？不是你的根，不是你的基因，机会就渺茫了。

    \

    什么造就你，什么也能毁灭你！
