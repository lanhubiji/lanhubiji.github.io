    VR家居：有内容才有未来

    {style=”text-align: justify; margin-left: 0px; text-indent: 0px;”}

    一、VR只炒概念无法持续 {style=”text-align: justify; margin-left: 0px; text-indent: 0px;”}

    \

    2016年，高盛发布了《VR与AR：解读下个通用计算平台》，它预测2025年整个市场营收达到800亿美元，其中硬件为450亿，软件为350亿。零售业将是虚拟现实技术的重要应用领域。

     

    VR是什么？VR就是虚拟现实，英文Virtual
    Reality的缩写。它脱胎于1980年代，借助计算设备和传感器技术创造出新型人机交互方式。它通过建模和仿真，在多维信息空间上创造出虚拟信息环境，让用户产生身临其境的感受。

     

    由于有独特的沉浸式体验，VR技术从2015年受到资本和创业者的追捧，恰似O2O的风口，但2016年下半年以来遭遇资本寒冬，概念降温。

     

    VR不仅有VR游戏、VR视频、VR教育、VR医疗等，更有VR家居，VR零售等。VR家居，对于人们家居设计有特别的吸引力，比如它营造出虚拟现实的环境让用户能够切身体验到最终效果。

     

    显然很多人都看到了其中的巨大商机，在百度上输入“VR家居”，可以看到很多厂商都已经积极想吃这股ＶＲ红利。但家居厂商在花了不少钱购买和搭建家居ＶＲ之后发现，并没有预想中的那么好，虽然比平面图逼真，但最终的实际效果和ＶＲ显示效果相差甚远。

     

    ＶＲ家居，看上去很美好，但离用户预期差距很远。做产品最重要的是用户体验，用户体验最重要的是心理预期。先不要说用户惊喜，超出预期，现在连符合预期的基本要求都不到。这会让很多用户失望，让家居厂商失望，最终ＶＲ家居产业无从发展。

     

    到底ＶＲ本身出问题了吗？

     

    二、VR本身没有问题，但体验远不够好 {style=”text-align: justify; margin-left: 0px; text-indent: 0px;”}

    \

    就拿VR应用最贴切的场景之一家居设计来说，如果用VR来看家居装修，对于用户来说，体验是极好的。

     

    从一个普通用户角度，不管是购房还是装修设计，看到一个平面化的图纸，不管你画的纸质效果图再精美，都很难让用户直观感受到整体空间。

     

    但如果有VR，可以立体地看到装修的风格、家具摆放，甚至可以考虑变换多个风格、多套家具，随意转化风格、空间关系等。还有就是选购家具时，用户可以带上VR眼镜，实现720度环视无死角的3D效果，可以感受到家具搬进家中的虚拟效果。家具在不同的环境下展示的效果不同，受到包括户型、空间、风格、搭配等众多因素影响，有了VR相当于提前把家具搬回家中进行模拟摆放，减少购买到不满意家居产品的概率。

     

    但，这些都是建立在VR技术的普及和硬件的完满解决以及有丰富内容的前提下。现在家居ＶＲ产品虚拟体验的感受不流畅，甚至还有眩晕感、定位不准，更重要的是VR家居缺乏大量的内容，没有足够的3D家居商品等。可以说，VR家居的前景很美好，但在此之前，只有概念是远远不够的，需要把概念落地才能迎来第二春。

     

    三、VR家居落地：什么问题是关键？ {style=”text-align: justify; margin-left: 0px; text-indent: 0px;”}

     

    对于ＶＲ家居的现状，有一家叫积木易搭的公司敏锐地捕捉到了其中的问题。现在的VR家居大多停留在概念上，卖个头盔眼镜，玩没几次用户就索然无味了。

     

    VR生态的核心还是要有足够的内容。不管是VR游戏、VR场景化应用，包括VR家居也是。你要有内容才行。

     

    记得2016年的双十一，阿里上线了VR购物平台“BUY+”，但更多还是一个概念的炒作，在它的整体销售占比中几乎可以忽略不计。最重要的原因之一是商品库还远远不够。要实现VR电商，还需要给所有的商品进行3D建模。一个商品的建模成本高达100多元，这个建模成本是不可承受的，有数十亿的商品，如果没有好的解决方案，也是无法迅速上架的。

     

    建模本身是个技术活，高质量的建模，一是要还原商品效果，还要恢复商品的材质，比如衣服的布料质地，甚至柔软和飘逸度、光影效果都得一一考虑并解决。要想大规模地形成商品库，没有快速且低成本的VR内容建模工具是不可能的。

     

    正是基于这样的考虑，积木易搭通过专注的研发，找到了一个快速建模的方式，并通过硬件和软件系统为众多家居厂商、家装设计企业提供快速建模的工具。

     

    积木易搭目前建立了低成本快速建模体系，可以迅速破解品牌商不愿意进行3D设计和VR体验的问题，快速建模能够抓住设计师和消费者的痛点，能够快速实现所见即所得，高度再现场景中的模型的真实尺寸、规格、款式、颜色、材料。一旦消费者与设计师探讨确定了设计图，即可进行购买和实施。

     

    积木易搭的建模系统能够做到10-20分钟内快速建模，把成本减低到目前行业成本的十分之一。它实现了采、编、存、发、用的全流程数据生产体系，实现了数据即服务、手机即服务的应用体系，并形成快速生存、众包发布中心、智能服务及云分享中心。

     

    四、解决VR家居内容：建模的突破 {style=”text-align: justify;”}

    \

    家居ＶＲ内容问题涉及到建模行业的成本、效率以及效果的问题。积木易搭的核心在于通过自己体系化的软硬结合方法破解了一个业界的难题：建模。传统的业界，建模并非不可能，但是成本高、时间长。没有人愿意去花这么大的成本去做这个事情。虽然3D效果好，用户体验好，但是成本高，周期长，无法落地。这也是VR家居业的最大问题。只要大规模低成本建模无法实现，家居VR就没有内容，也就是没有真正好的VR家居设计和选购体验。

     

    为了解决家居ＶＲ内容制造时间长、费用高、效果差的问题，积木易搭主要从以下几个方面入手。

     

    首先，确定行业的核心痛点

     

    目前整个家居ＶＲ行业，制造硬件设备比如头盔的不缺，做系统平台的不缺，唯独内容制造很缺。

     

    每年都有成千上万的家居产品，一个传统家居企业招人来做3D建模，做家居ＶＲ内容，基本上不太可能。一个衣柜，可能需要几天时间才能完成。如果几千件产品，一年都做不完，谁也耗不起这个时间：等你做完了更新换代又来了。

     

    不仅耗费时间，而且制作费用也高，效果也难如人意。传统的解决方案是使用3Dmax等设计软件进行人工建模，人工建模精度优于２厘米，即使采用三维扫描＋人工辅助，优于１毫米，整体效果还达不到高度仿真。另外，成本来说，根据难度不同，价格可以从200到2000元不等。一家家居企业几千件产品，成本也不小。

     

    从这个角度看，ＶＲ生态要建立起来看内容。ＶＲ内容要有好的体验，有大规模落地可能，主要看建模技术的发展。

     

    捕捉到了这个核心原因，积木易搭采用了集中优势兵力，Allin建模的方式去撬动整个行业的发展。

     

    这就是行业的杠杆。

     

    其次，根据痛点找到解决方案，实现质的提升

    \

    在找到行业发展的核心痛点之后，方向明确带来技术的提升。积木易搭创造属于自己的一整套建模技术。

    \

    一开始积木易搭的建模技术也不是最好的，有很多设备价格高，比如有的六七十万一台，积木易搭买的设备十七八万一台。创始人兼董事长孙剑峰为了提升效果，从国外购买回来当时全世界最好的设备，因为这个设备可以扫黑，可以扫高反光。效果是不错。但是后来发现两个问题比较难解决：一个是扫描出来容量大，一个小椅子就达到1Ｇ，大的甚至5个Ｇ，如果一个家居设计，动不动几十个产品，沙发、衣橱、床头柜等等，用户在移动端使用肯定超级慢。第二是从加拿大购买回来的６台设备，由于工作超负荷，很快就烧坏了３台。要维修就要寄到国外，然后再寄回来，一个多月时间就过去了。一天才能扫个二三十个，如果要把天猫上几亿商品扫完，需要上千年。

    \

    这个建模的效率和时间成本让孙剑峰感到崩溃。这个模式根本无法支撑起整个ＶＲ内容的制造，更无法支撑起ＶＲ家居的生态需求。

    \

    他决心自行研发，组织了顶尖专家，最后把国外卖270多万的厢式结构都做出来了。不仅成本是原来的十分之一，而且这个箱式结构15分钟内就能扫描完毕。目前它有六大硬件家族，包括自导机器臂自动扫描仪、厢式扫描仪、影像建模仪、立式扫描仪、手持白光扫描仪、手持激光扫描仪。这些三维建模完全自动，速度快、费用低、精度高。

    （箱式扫描仪）\

    不仅如此，这些硬件配套的数据生产体系可以快速把产品转化为数据。由于有了这套家居ＶＲ扫描解决方案，家居ＶＲ内容制作变得简单，方便，成本低，效果好。对于用户来说，0.0024厘米的精度，让设计师现场设计出来的效果图和最终的效果，几乎一样。

    （自导机械臂自动扫描仪）\

    \

    积木易搭思考是从整个生态系统的角度出发，不仅考虑如何建模，也考虑建模到底在什么场景下用。这样思考模式就把整个ＶＲ家居产业想通了。传统厂家拍照建模，一年上亿销量就需要200万左右的费用，而且很多手绘的３Ｄ的建模很难在ＶＲ上用。另外，关于材质渲染方面，积木易搭做到了真正的精模，每个材质的渲染参数不一样，每个模型都有参数。让用户从效果图看到的和最终的效果几乎一样。

    \

    最终，它能把建模时间从3-5天降低到３０分钟以内，从200-2000元的价格，降低到20-200元，精度从几厘米降低到0.0024厘米的精度，提升了数千倍。技术的极大提升带来了整个ＶＲ内容制作行业效率的极大提升，成本的极大降低以及效果的极大提升。

    \

    （所见即所得的实景与效果对比）

    \

    积木易搭通过建模把家居厂商、设计师、用户连接起来了。目前更像是一个为家居厂商、商家以及设计师提供的一个ＶＲ内容建模服务的企业。但由于厂家的商品都通过扫描进入到了它的商品库，最终所有的扫描建模的商品可以变成ＶＲ商城。

    \

    不同的厂家通过积木易搭的扫描建模硬件在VR上建立其VR商店，设计师为消费者设计家居空间，把VR商店的商品直接实现所见即所得的设计效果，一旦消费者满意，直接在VR商店下单支付。这样积木易搭就可通过扫描建模的入口把用户、设计师、厂家都串联起来，最终实现共赢。如果实现了这样的用户体验，最终这是一个巨大无比市场和巨大无比的生意。
